## 本集主軸：一條工作流，四個能力

> **讀進來 → 處理 → 抓資料 → 產出去**
>
> 這四步剛好構成一個完整迴圈。本集每個能力**只各示範一個**老師情境，當作「能力地圖」，讓你知道 Agent 的手腳能伸到哪裡。深入玩法後面集數再講。

| | 能力 | 一句話 | 老師現場 demo |
|---|------|--------|---------------|
| ① 讀 | 讀寫本地檔案 | 它看得到、改得動你硬碟裡的檔案 | 整理凌亂資料夾 + 批次改檔名 |
| ② 處理 | 執行程式與裝工具 | 它能寫程式、跑程式、裝工具 | 寫一段 Python 幫你算全班平均 |
| ③ 抓 | 上網與下載 | 它能自己上網查、自己下載 | 上網抓最新資料／下載素材 |
| ④ 產出 | 生成各種格式檔案 | 它能直接吐出成品檔 | 一句話產出 Word 考卷／Excel 成績表 |

## 開場：為什麼這集很關鍵？（3 分鐘）

- 回扣 EP01：生成式 AI（ChatGPT 網頁版）只能「跟你聊、給你文字」，你要自己複製貼上、自己存檔、自己整理。
- Agent 不一樣：它有「**手腳**」——能直接讀你電腦的檔案、能跑程式、能上網、能把成品存成檔案給你。
- 一句話定調：**「生成式 AI 動嘴，Agent 動手。」**
- 提醒：以下示範都建議在 EP02 教的「半自動」權限、限制在專案目錄下進行。

---

## 能力 ①：讀寫本地檔案（讀）

### 觀念
- Agent 看得到你指定資料夾裡的所有檔案，也能新增、修改、搬移、刪除、改名。
- 它能「讀懂」內容：PDF、Word、Excel、txt、圖片裡的文字都讀得到。
- 這是所有能力的地基——它要先「看得到你的檔案」，才能幫你做事。

### 現場 demo：整理一個凌亂的資料夾
- 準備一個亂七八糟的資料夾（混雜講義、考卷、圖片、影片）。
- 提示詞示範：
```
這個資料夾很亂，幫我：
1. 依檔案類型分到「講義」「考卷」「圖片」「影片」四個子資料夾
2. 把所有檔名統一改成「日期_主題」的格式
做之前先列一個計畫給我看，我說 OK 再動手。
```
- 鏡頭重點：示範它「先規劃 → 你確認 → 半自動執行」，呼應 EP02 的 Plan Mode 習慣。

### 口訣
> **它的眼睛和手，都在你開給它的那個資料夾裡。**

## 能力 ②：執行程式與裝工具（處理）

### 觀念
- Agent 不只會「寫」程式，它會「跑」程式，然後把結果給你——你完全不用懂程式。
- 需要的工具沒裝？它會用 `pip`（Python 套件）、命令列自己裝起來。
- 對老師的意義：批次運算、格式轉換、資料處理，這些「人工做很煩」的事，丟給它跑。

### 現場 demo：寫一段 Python 幫你算全班成績
- 準備一份成績 Excel（或 CSV）。
- 提示詞示範：
```
這份 grades.xlsx 是全班段考成績，幫我：
1. 算出每個人的平均、全班平均、最高最低分
2. 標出不及格的同學
3. 用 Python 跑，把結果存成一份新的 Excel
```
- 鏡頭重點：它會自己寫 Python、自己跑、（必要時）自己 `pip install` 缺的套件，最後丟成品給你。
- 進階補充（口頭帶過）：複雜工具也能裝，例如裝 ffmpeg 把影片轉成音檔——點到為止，當作能力上限的展示。

### 口訣
> **你不用會寫程式，因為它會寫、也會跑。**

## 能力 ③：上網與下載（抓）

### 觀念
- Agent 能自己開網路、讀網頁內容、抓你要的資料，也能把檔案下載到本機。
- 跟生成式 AI 的差別：它抓的是「**現在**」的網路資料，不是訓練時的舊記憶。
- 對老師的意義：找最新時事入題、查資料、下載素材，不用自己一個個開分頁。

### 現場 demo：上網查資料 / 下載素材
- 提示詞示範（擇一）：
```
上網幫我查「今年的某個時事數據」，整理成 3 個重點，附上資料來源連結。
```
或
```
幫我到某個免費圖庫，下載 5 張「適合放在數學簡報」的背景圖，存到 slides/ 資料夾。
```
- 鏡頭重點：示範它「自己上網 → 讀內容 → 抓回來 → 存檔」的完整動作。
- 安全提醒：上網與下載屬於 EP02 講的敏感權限，新手保持「需確認」，看清楚它要抓的網址再放行。

### 口訣
> **它上的是現在的網，抓回來的是真檔案。**

---

## 能力 ④：生成各種格式檔案（產出）

### 觀念
- 前三個能力都在「輸入端」，這個是「**輸出端**」——把成果變成你能直接用的檔案。
- 它能產出：Word、Excel、PPT、圖片、圖表、HTML 網頁⋯⋯
- 對老師的意義：考卷、成績表、簡報、教學圖表，一句話就有成品檔，不是只給你一段文字要你自己排版。

### 現場 demo：一句話產出一份 Word 考卷
- 提示詞示範：
```
幫我出一份國中數學「一元一次方程式」小考：
- 10 題，難度由淺到深，最後 2 題是應用題
- 附詳解
- 直接做成一份 Word 檔，題目卷和答案卷分開
```
- 鏡頭重點：成品直接是 .docx，打開就能印，呼應「動手」而不是「動嘴」。

### 口訣
> **生成式 AI 給你文字，Agent 給你成品檔。**

## 收尾 1：安全提醒（必講，篇幅小）

- 這集第一次完整展示 Agent 能對你電腦「動手」的威力——能力越大，越要回扣 EP02 的權限觀念。
- 三個原則：
  1. **半自動為主**：重要動作讓它先規劃、你確認再執行。
  2. **限制在專案目錄**：別讓它在整顆硬碟亂跑。
  3. **來路不明的工具/Skill 不要亂裝**：尤其會上網、會下載、會裝套件的時候。
- 一句話：**「它比我們可靠，但別把方向盤完全放掉。」**

## 收尾 2：下集預告（各 60 秒）

- **EP04（預告）：連接外部工具（MCP）**——讓 Agent 不只動本機檔案，還能接你的 Google Drive、Gmail、Classroom、Obsidian，真正接管你的數位工作。
- **EP05（預告）：技能（Skill）**——把本集這些能力「打包成一句話」自動觸發，做出你自己的專屬工作流。
