## 🚀 一鍵安裝所有工具（重點！）

把 [`AGENT_SETUP_教學檔案處理工具包.md`](AGENT_SETUP_教學檔案處理工具包.md) 交給你的 AI Agent
（Claude Code / Codex / Antigravity / OpenCode 都可以），然後說：

> 「讀這份檔案，幫我把裡面的 Python 套件都安裝好。」

你的 Agent 就會自動偵測環境、安裝套件、處理系統相依並回報結果。
