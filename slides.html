觀念都懂了，那它到底能幫我做什麼？這集開始，我們看 Agent 真正「動手」。
本集定調

生成式 AI 動嘴，Agent 動手

💬 生成式 AI（網頁聊天）

  • 只能跟你「聊」、給你文字
  • 要你自己複製貼上
  • 要你自己存檔、自己整理
  • 碰不到你電腦裡的檔案

🦾 AI Agent（有手腳）

  • 直接讀、改你電腦的檔案
  • 自己寫程式、跑程式
  • 自己上網查、自己下載
  • 把成品直接存成檔案給你

這集是 EP01 那句「Agent 能動手」的兌現集。

能力 ① ・ 讀

讀寫本地檔案：所有能力的地基

👀
看得到
你開給它的資料夾，裡面所有檔案它都看得到。
📖
讀得懂
PDF、Word、Excel、txt、圖片裡的文字，都讀得進來。
✏️
改得動
新增、修改、搬移、刪除、改名，全部能做。
它要先「看得到你的檔案」，才能幫你做事 — 這是後面三個能力的前提。
能力 ① ・ 現場 demo

整理一個凌亂的資料夾

# 對著一個亂七八糟的資料夾說： 這個資料夾很亂，幫我： 1. 依檔案類型分到「講義」「考卷」「圖片」「影片」四個子資料夾 2. 把所有檔名統一改成「日期_主題」的格式 # 關鍵：做之前先列計畫，我說 OK 再動手 做之前先給我一個計畫，我確認再執行。

鏡頭重點：示範「先規劃 → 你確認 → 半自動執行」，呼應 EP02 的 Plan Mode 習慣。

口訣：它的眼睛和手，都在你開給它的那個資料夾裡。

能力 ② ・ 處理

執行程式與裝工具：你不用會寫程式

✍️
會寫程式
你用自然語言講，它幫你把 Python 寫出來。
▶️
會跑程式
寫完直接執行，把結果交給你 — 你不用碰程式。
📦
會裝工具
缺套件就用 pip 自己裝；複雜工具（如 ffmpeg）也裝得起來。
對老師的意義：批次運算、格式轉換、資料處理這些「人工做很煩」的事，丟給它跑。
能力 ② ・ 現場 demo

寫一段 Python 幫你算全班成績

# 準備一份成績 Excel，然後說： 這份 grades.xlsx 是全班段考成績，幫我： 1. 算出每個人的平均、全班平均、最高最低分 2. 標出不及格的同學 3. 用 Python 跑，把結果存成一份新的 Excel

鏡頭重點：它自己寫 Python、自己跑、必要時自己 pip install 缺的套件，最後丟成品給你。完全不用你懂程式。

口訣：你不用會寫程式，因為它會寫、也會跑。

能力 ③ ・ 抓

上網與下載：它抓的是「現在」的網路

🌐
自己上網
開網路、讀網頁、抓你要的資料，不用你一個個開分頁。
🆕
抓最新
和生成式 AI 不同 — 它抓「現在」的網路，不是訓練時的舊記憶。
⬇️
能下載
把圖片、檔案、素材直接下載到你的本機資料夾。
對老師的意義：找最新時事入題、查資料、下載素材，一句話搞定。
能力 ③ ・ 現場 demo

上網查資料 / 下載素材

# 查最新資料： 上網幫我查「今年的某個時事數據」， 整理成 3 個重點，附上資料來源連結。 # 或下載素材： 幫我到免費圖庫，下載 5 張「適合放在數學簡報」的背景圖， 存到 slides/ 資料夾。

鏡頭重點：示範「自己上網 → 讀內容 → 抓回來 → 存檔」的完整動作。

⚠️ 上網與下載是敏感權限：新手保持「需確認」，看清楚要抓的網址再放行。口訣：它上的是現在的網，抓回來的是真檔案。

能力 ④ ・ 產出

生成各種格式檔：給你的是成品，不是文字

📄
Word
考卷、講義、通知單
📊
Excel
成績表、統計表
📑
PPT / 圖表
簡報、教學圖表
🌐
圖片 / HTML
插圖、互動網頁
前三個能力都在「輸入端」，這個是「輸出端」 — 把成果變成你打開就能用的檔案。
能力 ④ ・ 現場 demo

一句話產出一份 Word 考卷

# 直接說： 幫我出一份國中數學「一元一次方程式」小考： - 10 題，難度由淺到深，最後 2 題是應用題 - 附詳解 - 直接做成一份 Word 檔，題目卷和答案卷分開

鏡頭重點：成品直接是 .docx，打開就能印 — 這就是「動手」而不是「動嘴」。

口訣：生成式 AI 給你文字，Agent 給你成品檔。

收尾 1 ・ 安全

能力越大，越要回扣權限觀念

🤝
半自動為主
重要動作讓它先規劃、你確認再執行。
📁
限制在專案目錄
別讓它在整顆硬碟亂跑。
🚫
不裝來路不明的東西
尤其會上網、會下載、會裝套件的時候。
一句話：它比我們可靠，但別把方向盤完全放掉。
